# 🚗 Automotive Sales & Service Analytics Dashboard

## 📌 Project Overview

This project focuses on analyzing **automotive sales and service data** using **Microsoft Excel** and **IBM Cognos Analytics** to create interactive visualizations and dashboards that support business decision-making.

The project is divided into two parts:

* **Part 1:** Sales Analysis using Microsoft Excel
* **Part 2:** Interactive Sales & Service Dashboard using IBM Cognos Analytics

The objective is to help regional managers monitor dealership performance, evaluate profitability, analyze customer service metrics, and identify trends across different car models and dealers.

---

# 🛠️ Tools & Technologies Used

* Microsoft Excel (Excel for Web/Desktop)
* IBM Cognos Analytics
* Pivot Tables
* Pivot Charts
* Dashboard Design
* Data Visualization
* KPI Analysis
* Business Intelligence

---

# 📊 Part 1: Sales Analysis in Microsoft Excel

## Project Objective

Create meaningful visualizations from dealership sales data to understand sales performance, profitability, and dealer-wise trends.

## Dataset

The dataset contains automobile dealership sales information including:

* Dealer ID
* Car Model
* Quantity Sold
* Profit
* Sales Date
* Year

## Visualizations Created

### 📈 Quantity Sold by Dealer ID

* Bar Chart
* Sorted in ascending/descending order
* Displays dealer-wise sales performance.

### 📉 Profit by Date and Model

* Line Chart
* Shows profit trends over time for different vehicle models.

### 📊 Profit by Year and Dealer ID

* Column Chart
* Customized with red-colored columns
* Compares yearly profit generated by each dealer.

### 📍 Profit of Hudson Models by Dealer ID

* Line Chart
* Horizontal gridlines removed
* Legend positioned on the right
* Green series outline for improved visualization

# 📊 Part 2: IBM Cognos Analytics Dashboard

## Project Objective

Develop an interactive dashboard that provides insights into both **Sales** and **Service** operations across automobile dealerships.

Two dashboards were created:

* **Sales Dashboard**
* **Service Dashboard**

---

# 📈 Sales Dashboard

The Sales dashboard presents key business metrics through multiple KPI panels.

### KPI Cards

* Profit (formatted in millions of USD)
* Quantity Sold
* Quantity Sold by Model
* Average Quantity Sold

### Dealer Performance Visualization

* Profit by Dealer ID
* Displayed as a column chart
* Sorted in ascending order for easier comparison

---

# 🔧 Service Dashboard

The Service dashboard focuses on after-sales performance and customer satisfaction.

## Visualizations Included

### 🚘 Number of Recalls per Model

* Column Chart
* Displays recall frequency for each vehicle model.

### 😊 Customer Sentiment Analysis

* Treemap
* Compares positive, neutral, and negative customer reviews.

### 📅 Monthly Quantity Sold vs Profit

* Combined Line and Column Chart
* Helps analyze the relationship between sales volume and profitability over time.

### 🔥 Recalls by Model and Affected System

* Heat Map
* Highlights models and systems with unusually high recall counts.

# 🔍 Key Business Insights

* Sales performance varies significantly across different dealers.
* Certain vehicle models contribute disproportionately to total profits.
* Customer sentiment provides valuable feedback for service improvement.
* Recall patterns help identify recurring issues with specific models or systems.
* Comparing monthly sales with profit enables better forecasting and strategic planning.
* KPI dashboards provide quick access to critical business metrics for management.

# 💼 Business Value

The dashboards enable managers to:

* Monitor dealership performance in real time.
* Identify high-performing and underperforming dealers.
* Track profitability trends across models and years.
* Analyze customer satisfaction through sentiment analysis.
* Detect recall-related issues for proactive maintenance.
* Make data-driven decisions using interactive visualizations.
